Fullstack Monorepo Starter

About

A full stack Monorepo starter with Angular, GraphQL & Kubernetes.

Guides to clone, re-brand and deploy your own can be found here

Whats in the box?

  • An Angular Todo application with NgRx & GraphQL including user registration and authentication.
  • Node GraphQL API with Koa, GraphQL & MongoDB.
  • Guides to deploying your Angular App with Firebase Hosting or SSR with Firebase Functions & your API on Google Cloud Kubernetes engine.
  • More coming soon!

Bonus: examples of some less common/more advanced Angular techniques
