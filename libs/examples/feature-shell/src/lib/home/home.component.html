Zero to Production

Fullstack Monorepo Starter

About

A full stack Monorepo starter to use for the foundation for your next project.

Guides

Follow the guides to step through deploying your own UI with Firebase and your API either serverless with AWS Lambda or in a Kubernetes Cluster on Google Cloud.

Whats in the box

  • Two Angular apps. This "examples" demo site and a Todo App with authentication using NgRx & GraphQL
  • A Node.js GraphQL API with Koa, GraphQL & MongoDB configured to deploy serverless or with Docker.
  • Guide to get you up and running with a deployed project.

Angular Examples

Examples of some less common/more advanced Angular techniques.
