Zero to Production

Fullstack Monorepo Starter

About

An example of a full stack Monorepo with Angular, GraphQL & Kubernetes.

Instructions to re-brand and get you up an running with your own deployed client and server can be found here.

Whats in the box?

  • An Angular Todo application with NgRx & GraphQL including user registration and authentication.
  • Node API with Koa, GraphQL & MongoDB (including a custom Angular CLI Builder).
  • Guides to deploy you server with Traefik revers proxy Google Cloud Kubernetes engine.
  • More coming soon

Bonus

Demonstrations of some less common/more advanced Angular techniques
