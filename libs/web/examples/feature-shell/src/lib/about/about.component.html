About

The goal of this project is to demonstrate a full stack Monorepo as well as providing the foundation for you to start your own (plus having fun along away)

Whats in the box?

  • A couple of frontend applications including this 'examples' site and a Todo Application with the whole box and dice (Auth, registration, master-detail etc)!
  • Live examples of some less common Anglar techniques.
  • Node API using Koa, GraphQL and MongoDB (including a custom Angular CLI Builder)
  • Deploy you server in a Kubernetes environnement on Google Cloud
  • Much more to come
